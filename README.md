# FerreraFreight

FerreraFreight es una organización de desarrollo orientada a construir herramientas internas para automatización logística, integración de datos, reporting operativo y asistencia mediante agentes.

## Áreas principales

* **Reporting operativo:** visualización de KPIs, excepciones, estados y cuellos de botella.
* **Datos y ETL:** carga, limpieza, normalización y validación de información logística.
* **Base de datos:** modelo relacional, migraciones, esquemas, constraints y auditorías.
* **Agentes y automatización:** flujos n8n, parsers, integración con canales de comunicación y automatización de tareas operativas.

## Repositorios principales

* `logistica-reporting` — aplicación de reporting y consulta operativa.
* `logistica-etl` — extracción, limpieza y preparación de datos.
* `logistica-bbdd` — esquema de base de datos, migraciones y validaciones.
* `logistica-agent` — workflows, parsers y automatización del agente logístico.

## Política de trabajo

Los repositorios activos no deben contener credenciales, datos reales, backups operativos ni artefactos locales.
La colaboración se realiza mediante ramas, pull requests y revisión antes de integrar cambios en `main`.

## Estado

Proyecto en fase de consolidación técnica y preparación para colaboración controlada.
